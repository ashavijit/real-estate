# Clone the Project
```

    git clone <repo>
    cd repo

```
# Install the Project
```
    npm install
```
# Build the webpack
```
    npm run build
```

# Run the server
```
    npm run dev
```

# It will start a server on port 7800
# You can access the server at http://localhost:7800

# Run the tests
```
    npm run test
```
